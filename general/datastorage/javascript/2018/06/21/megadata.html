Storing Massive Files in a Small Space

June 21st, 2018

Have you ever wanted to store a massive file of the same text over and over again? If so boy do I have the solution for you. It is called MegaFile.

What is it?

MegaFile is Nasa Grade Software written in NodeJS. This software with its Nasa grade compession allows for the user to convert a tiny file of < 30 Bytes into a multi Gatobyte file!

How does it work?

MegaFile uses the magic of javascript, ai, complex math, and a chain of a single text block (blockchain) to convert these tiny files into massive ones.

Use Case?

Think of all the fame and glory you could have in your friend group. While everyone else is bragging about their multi-terrabyte gaming PCs for 20TB nases you can brag about storing 1 Gatobyte on your diskette from the 80s!

More Info?

More info can be found at MegaFile. That link includes instructions on making the massive files and how to decompress them
